`Сколько времени обучающийся будет находиться в школе, учась со 2-го по 4-ый уч. предмет?`

```sql
SELECT SEC_TO_TIME(
		TIME_TO_SEC(
			(
				SELECT end_pair
				FROM Timepair
				WHERE id = 4
			)
		) - TIME_TO_SEC(
			(
				SELECT start_pair
				FROM Timepair
				WHERE id = 2
			)
		)
	) AS time;
```
